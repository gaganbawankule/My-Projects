# Social Media Usage Analytics Dashboard (Power BI)

## Project Overview

This Power BI project analyzes how students' social media usage affects academic performance, sleep patterns, mental health, and relationship dynamics.

The dashboard was developed using Power BI to transform raw student and platform usage data into meaningful business insights through interactive visualizations, drill-through reports, bookmarks, and slicers.

## Objectives

* Analyze social media usage patterns among students
* Measure the impact of social media on academic performance
* Study the relationship between screen time and mental health
* Evaluate sleep habits and addiction levels
* Create interactive dashboards for stakeholder decision-making

## Key Insights

* Social media usage patterns vary across academic levels.
* Higher addiction scores are associated with lower mental health scores.
* Sleep duration shows a strong relationship with social media engagement.
* Academic performance is influenced by daily social media usage habits.
* Platform preferences differ significantly by demographic groups.

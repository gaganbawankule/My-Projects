# Customer Insights - A Statistical Investigation

## Overview

Performed statistical analysis on customer demographic and spending data using Python. Applied descriptive statistics, exploratory data analysis (EDA), visualizations, and hypothesis testing to uncover customer behavior patterns and business insights.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

## Skills

- Descriptive Statistics
- Exploratory Data Analysis (EDA)
- Data Visualization
- Hypothesis Testing
- T-Test
- ANOVA
- Correlation Analysis
- Customer Segmentation
- Statistical Reporting

## Project Files

- Stats_mini_project.ipynb
- Project_Assessment.pdf

## Key Objectives

- Analyze customer demographics and spending behavior
- Visualize customer trends and distributions
- Compare customer groups statistically
- Test business hypotheses using statistical methods
- Generate actionable business insights

## Dataset

Customer dataset containing demographic information, spending behavior, education level, marital status, pet ownership, and customer engagement metrics.

# Real Estate Insights Dashboard (Tableau)

## Project Overview

This Tableau project analyzes real estate sales transactions from 2011–2022 to uncover trends in property values, sales performance, and regional market activity.

The dashboard was developed for decision-makers to better understand sales patterns, property distributions, and town-level performance through interactive visualizations.

## Objectives

* Analyze real estate sales trends over time
* Compare property values across towns
* Evaluate sales ratios and transaction volumes
* Identify top-performing locations
* Provide interactive insights for decision-making

## Dataset Information

The dataset contains real estate transaction records from 2011–2022.

Key attributes include:

* Town
* Property Type
* Residential Type
* Sale Amount
* Assessed Value
* Sales Ratio
* Date Recorded
* State
